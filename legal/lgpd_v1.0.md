# Política de Proteção de Dados (LGPD)

**Meu Condomínio Fácil**  
**CNPJ: 57.444.727/0001-85**

**Versão 1.0 | Vigência: 17/12/2025**

---

## 1. INTRODUÇÃO

A **Meu Condomínio Fácil**, pessoa jurídica de direito privado, inscrita no CNPJ sob o nº **57.444.727/0001-85**, na qualidade de **Controladora de Dados**, compromete-se a proteger a privacidade e os dados pessoais de seus usuários em conformidade com a Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018 - LGPD).

Esta Política descreve como coletamos, usamos, armazenamos, compartilhamos e protegemos seus dados pessoais, bem como seus direitos como titular desses dados.

---

## 2. DEFINIÇÕES

Para fins desta Política:

- **LGPD**: Lei Geral de Proteção de Dados Pessoais (Lei nº 13.709/2018).
- **Controlador**: Meu Condomínio Fácil, pessoa jurídica que toma as decisões sobre o tratamento de dados pessoais.
- **Operador**: Quando o Condomínio (cliente) utiliza nossa plataforma para tratar dados de seus moradores.
- **Titular**: Pessoa natural a quem se referem os dados pessoais.
- **Tratamento**: Toda operação realizada com dados pessoais (coleta, armazenamento, uso, compartilhamento, exclusão).
- **Dados Pessoais**: Informação relacionada a pessoa natural identificada ou identificável.
- **Dados Sensíveis**: Dados sobre origem racial, convicção religiosa, saúde, vida sexual, dados genéticos ou biométricos.

## 4. COMO USAMOS SEUS DADOS

Utilizamos os dados pessoais para:

1. **Prestação do serviço contratado**
   - Autenticação e acesso ao sistema
   - Gestão de unidades, moradores e cobranças
   - Controle de portaria e visitantes
   - Envio de notificações (avisos, cobranças, encomendas)

2. **Comunicação**
   - E-mails sobre sua conta (ativação, cobrança, suporte)
   - Notificações operacionais do sistema
   - Comunicados importantes sobre o serviço

3. **Cumprimento de obrigações legais**
   - Emissão de notas fiscais
   - Atendimento a requisições da ANPD ou autoridades

4. **Melhoria do serviço**
   - Análise de uso para otimização
   - Correção de bugs e melhorias

5. **Segurança**
   - Prevenção de fraudes
   - Logs de acesso
   - Proteção contra ataques

## 5. COMPARTILHAMENTO DE DADOS

Compartilhamos dados pessoais apenas nas seguintes situações:

### 5.1 Prestadores de Serviço (Operadores)

| Prestador | Finalidade | Dados Compartilhados |
|-----------|------------|---------------------|

### 5.2 Integrações Opcionais

Quando contratadas pelo cliente:

| Serviço | Finalidade | Dados |
|---------|------------|-------|
| Evolution API / WhatsApp | Notificações via WhatsApp | Telefone, mensagem |
| Asaas/PagSeguro/Outros | Cobranças bancárias | CPF, nome, valor |

> **Nota**: Estas integrações são contratadas separadamente e o cliente é responsável por obter consentimento dos moradores para recebimento de mensagens.

### 5.3 Autoridades

Podemos compartilhar dados quando exigido por lei, ordem judicial ou requisição de autoridade competente.

## 6. SEUS DIREITOS (LGPD Art. 18)

Como titular de dados, você tem direito a:

| Direito | Descrição |
|---------|-----------|
| **Confirmação** | Saber se tratamos seus dados |
| **Acesso** | Obter cópia dos seus dados |
| **Correção** | Corrigir dados incompletos ou incorretos |
| **Anonimização** | Solicitar anonimização quando possível |
| **Portabilidade** | Receber seus dados em formato estruturado |
| **Eliminação** | Solicitar exclusão de dados tratados com consentimento |
| **Revogação** | Revogar consentimento a qualquer momento |
| **Informação** | Saber com quem compartilhamos seus dados |
| **Oposição** | Opor-se a tratamento em certas situações |

### Como Exercer Seus Direitos

Envie solicitação para: **privacidade@meucondominiofacil.com**

Prazo de resposta: até 15 dias úteis.

## 7. RETENÇÃO DE DADOS

| Tipo de Dado | Período de Retenção |
|--------------|---------------------|
| Dados de conta ativa | Durante a vigência do contrato |
| Dados após cancelamento | 5 anos (obrigações fiscais) |
| Logs de acesso | 6 meses |
| Dados de visitantes | 2 anos (segurança) |
| Dados de pagamentos | 5 anos (obrigação fiscal) |

Após os períodos acima, os dados são anonimizados ou eliminados.

## 8. SEGURANÇA DOS DADOS

Implementamos medidas técnicas e administrativas para proteger seus dados:

### Medidas Técnicas
- ✅ Criptografia em trânsito (HTTPS/TLS)
- ✅ Criptografia em repouso (AES-256)
- ✅ Row Level Security (RLS) no banco de dados
- ✅ Autenticação segura com tokens
- ✅ Backups automáticos diários
- ✅ Monitoramento de acessos

### Medidas Administrativas
- ✅ Acesso restrito a dados (need-to-know)
- ✅ Política de senhas fortes
- ✅ Treinamento em proteção de dados
- ✅ Contratos com operadores

---

## 11. MENORES DE IDADE

Nosso serviço não é destinado a menores de 18 anos. Dados de menores só são tratados quando necessário para o serviço (ex: moradores menores cadastrados por seus responsáveis).

---

## 12. PAPEL DO CONDOMÍNIO (CLIENTE)

Quando o Condomínio utiliza nossa plataforma para gerenciar dados de moradores:

- O **Condomínio** atua como **Controlador** dos dados dos moradores
- A **Meu Condomínio Fácil** atua como **Operadora**
- O Condomínio é responsável por:
  - Obter consentimento dos moradores quando necessário
  - Informar moradores sobre o tratamento de dados
  - Atender solicitações de direitos dos moradores

---

## 13. INCIDENTES DE SEGURANÇA

Em caso de incidente de segurança que possa acarretar risco ou dano aos titulares:

1. Comunicaremos a ANPD em prazo razoável
2. Notificaremos os titulares afetados
3. Tomaremos medidas para mitigar danos
